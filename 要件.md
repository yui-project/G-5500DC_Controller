# 概要
- G-5500DCを、Arduino Mega 2560をインターフェースとしてPCから操作するためのソフトウェア
- Mac, Windowsの両方で使用可能
- 制御ソフトウェアはモダンなデザインに
- ArduinoはPlatform IO環境で使用し、パソコンとはUSBで接続

# G-5500DCについて
## ピン対応
| 端子名 | 端子番号 | 内容 |
| --- | --- | --- |
| 水平回転角度検出端子 | 6 | 0～450°を 0V～(2.0V～4.5V)の電圧値に変換される。<br>出力インピーダンス 10kΩ |
| 垂直回転角度検出端子 | 1 | 0～180°を 0V～(2.0V～4.5V)の電圧値に変換される。<br>出力インピーダンス 10kΩ |
| LEFT回転制御端子 | 4 | 端子番号8と接続すると、ローターは左回転する。 |
| RIGHT回転制御端子 | 2 | 端子番号8と接続すると、ローターは右回転する。 |
| DOWN回転制御端子 | 5 | 端子番号8と接続すると、ローターはDOWN方向へ回転する。 |
| UP回転制御端子 | 3 | 端子番号8と接続すると、ローターはUP方向へ回転する。 |
| 出力電圧：約DC13V～8V<br>出力電流：100mA以下 | 7 | この電圧は定電圧ではありません。出力電流によって出力電圧は変動します。<br>100mA流した時利用できる電圧は、8V程度になります。（0.5Aのヒューズ内蔵） |
| アース端子 | 8 | - |

## Arduino Mega 2560 との接続関係
| G-5500DCの端子名 | Arduinoのピン |
| --- | --- |
| 水平回転角度検出端子 | A0 |
| 垂直回転角度検出端子 | A1 |
| LEFT回転制御端子 | 直接接続ではダメっぽい（電圧が20V近くあるため） |
| RIGHT回転制御端子 | 直接接続ではダメっぽい（電圧が20V近くあるため） |
| DOWN回転制御端子 | 直接接続ではダメっぽい（電圧が20V近くあるため） |
| UP回転制御端子 | 直接接続ではダメっぽい（電圧が20V近くあるため）｀ |
| 出力電圧：約DC13V～8V<br>出力電流：100mA以下 | - |
| アース端子 | GND |
